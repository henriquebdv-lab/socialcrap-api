# 💩 SocialCrap API

API REST desenvolvida em .NET 8 com Entity Framework Core para simular uma rede social simples inspirada no Orkut.

## 🚀 Funcionalidades

- 👤 Cadastro de usuários
- 💬 Criação de "Craps" (posts)
- 💩 Sistema de likes (Poop)
- 📰 Sistema de notícias (em desenvolvimento)

## 🧱 Tecnologias

- .NET 8
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Swagger
